# WebGados Demo
App developed using the mocked [WebGados API](https://webgados.com.br/anuncios-example) to show a list of public ads and open a detail page for a selected one.
# What was used?
  - Kotlin
  - MVP
  - Retrofit
  - Picasso
  - GSON
  - Room
  - 2 layouts based on device orientation
  - Material Design
